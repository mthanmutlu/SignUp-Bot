- Pythonu kurmak için `https://www.python.org/ftp/python/3.9.9/python-3.9.9-amd64.exe`

- Paketleri yüklemek için aşağıdaki komutu girin.
    pip install -r requirements.txt

- Botu çalıştırmak için `bot.py` dosyasını çalıştırınız.

- Hoş geldin mesajı üzerindeki resim URL' sini değişmek için `config.json` dosyasından güncelleyebilirsin.

- Bot tokenini değişmek için .env dosyasındanki `TOKEN` kısmını güncelleyebilirsin.

- Etkileşim sonrası verilmesini istediğin rolleri düzenlemek için `config.json` dosyasından listeye ekleme yapabilirsin
    Örn;
        "roles": [
            944566541817639003,
            129854125901285125,
        ]
